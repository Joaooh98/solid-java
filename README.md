# Projeto de Exemplificação dos Princípios SOLID em Java

## Introdução

Este projeto tem como objetivo exemplificar a aplicação dos princípios SOLID no desenvolvimento de software utilizando a linguagem Java. Os princípios SOLID são diretrizes de design que visam criar código mais flexível, adaptável e de fácil manutenção.

## Princípios SOLID

### 1. Princípio da Responsabilidade Única (SRP)

- **Descrição:** Cada classe deve ter apenas uma responsabilidade. Isso promove a coesão, facilita a manutenção e reduz a probabilidade de erros.
- **Exemplo no Projeto:** [Link para exemplo SRP](solid-java/src/principio-responsabilidade-unica)

### 2. Princípio do Aberto/Fechado (OCP)

- **Descrição:** Uma classe deve ser aberta para extensão, permitindo novos comportamentos sejam adicionados, mas fechada para modificação, evitando alterações em seu código existente. Isso é alcançado por meio do uso de abstrações e interfaces.
- **Exemplo no Projeto:** [Link para a Classe que Exemplifica o OCP]()

### 3. Princípio da Substituição de Liskov (LSP)

- **Descrição:** Os objetos de uma classe base devem poder ser substituídos por objetos de suas classes derivadas sem afetar a corretude do programa. As classes derivadas devem ser substituíveis por suas classes base sem introduzir comportamentos inesperados.
- **Exemplo no Projeto:** [Link para a Classe que Exemplifica o LSP]()

### 4. Princípio da Segregação de Interfaces (ISP)

- **Descrição:** Uma classe não deve ser obrigada a implementar interfaces que ela não utiliza. As interfaces devem ser específicas para as necessidades das classes que as implementam, evitando interfaces monolíticas.
- **Exemplo no Projeto:** [Link para a Classe que Exemplifica o ISP]()

### 5. Princípio da Inversão de Dependência (DIP)

- **Descrição:** Classes de alto nível não devem depender de classes de baixo nível, mas sim de abstrações. Abstrações não devem depender de detalhes, mas sim os detalhes devem depender das abstrações. Isso promove flexibilidade e modularidade.
- **Exemplo no Projeto:** [Link para a Classe que Exemplifica o DIP]()
